Algorytm wyszukiwania wśród elementów listy elementu o najmniejszej wartości. Przechodzimy po liście i porównujemy wartości poszczególnych elementów ze zmienną, do której na początku przypisujemy wartość pierwszego elementu. Jeżeli nowy element jest mniejszy od zmiennej, to zmieniamy wartość, w innym przypadku przechodzimy do kolejnej komórki listy.
